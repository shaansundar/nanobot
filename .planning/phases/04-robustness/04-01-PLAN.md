<objective>
Harden ClaudeCodeProvider subprocess lifecycle for production reliability: process group management with zombie prevention, semaphore-based concurrency limiting, and environment isolation for gateway mode.

Purpose: Without this hardening, sustained use can accumulate zombie processes (ROBU-01), unbounded concurrent subprocesses can exhaust host resources (ROBU-02), and gateway mode can leak API keys to channel users (ROBU-03).

Output: Hardened `_run_cli()` method, extended config schema, and comprehensive test suite covering all three robustness requirements.
</objective>

        async with self._subprocess_semaphore:  # D-08, ROBU-02

            try:

            except asyncio.TimeoutError:
                await self._kill_process_group(proc)  # D-03
                return LLMResponse(
